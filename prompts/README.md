# Prompts for AI-Assisted Customization

This folder contains context and prompts to help AI assistants customize the Arcade Composable Dashboard.

## Files

- **`llm.txt`** - Complete project context for AI assistants (Claude, GPT-4, etc.)
- **`arcade-api-reference.md`** - Quick reference for Arcade API endpoints
- **`customization-prompts.md`** - Pre-written prompts for common customization tasks

## How to Use

### 1. Give AI the Context

Share `llm.txt` with your AI assistant to give it complete project understanding:

```
I'm working on the Arcade Composable Dashboard. Here's the project context:
[Paste contents of llm.txt]
```

### 2. Use Pre-Written Prompts

Browse `customization-prompts.md` for common tasks and copy the relevant prompt.

### 3. Customize

The AI will make changes while maintaining:
- Code quality
- Consistent styling
- Type safety
- Project architecture

## Example Workflow

**Task**: Change branding to your company

1. Open `customization-prompts.md`
2. Find "Change Company Branding" prompt
3. Fill in your company details
4. Send to AI assistant
5. AI updates all necessary files

## Benefits

- **Faster customization** - Pre-written prompts save time
- **Consistent changes** - AI knows the project structure
- **Type-safe** - AI maintains TypeScript types
- **Best practices** - AI follows project patterns

## For Forward-Deployed Engineers

These prompts are especially useful for Arcade's forward-deployed engineers helping customers customize the dashboard. Use them as starting points and modify as needed.

## Adding New Prompts

When you create a useful customization, add it to `customization-prompts.md` so others can benefit.
