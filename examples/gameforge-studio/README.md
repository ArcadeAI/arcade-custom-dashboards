# GameForge Studio Dashboard

Gaming industry dashboard with vibrant purple/orange gradients and modern gaming aesthetics.

## Features

- Purple & Orange color scheme
- Dark theme with vibrant gradients
- Interactive analytics with hover tooltips
- Real-time tool execution tracking
- Gaming-focused branding

## Tech Stack

- Next.js 16
- Tailwind CSS
- TypeScript
- Lucide Icons

## Run This Example

```bash
# From the root directory
docker-compose up -d

# Or navigate here
cd examples/gameforge-studio
npm install
npm run dev
```

Visit: http://localhost:3000/examples/gameforge
